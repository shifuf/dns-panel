# DNS Panel 部署文档

详细部署、首次安装、Docker 启动、数据库自动初始化、备份与升级说明，统一请直接参考根目录 `README.md`。

当前项目已经移除历史上的 `prisma/` 与 `data/` 部署结构，SQLite 默认位置统一为：

```bash
backend/db/database.db
```

如果你是从旧版本迁移，请重点确认以下事项：

- `.env` 中的 `DATABASE_URL` 应改为 `file:/app/db/database.db`
- `docker-compose.yml` 仅挂载 `./backend/db:/app/db`
- 升级与备份脚本默认读取 `backend/db/database.db`
- 首次启动时无需预置数据库文件，后端会自动创建并初始化
